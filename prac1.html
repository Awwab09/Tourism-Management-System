About BCA Department

The BCA Department offers a three-year Bachelor of Computer Applications (BCA) programme that provides a strong foundation in computer science and software development. The course curriculum covers various areas such as programming languages, database management, web development, and software engineering.

Our Courses

  • Introduction to Programming
  • Data Structures and Algorithms
  • Database Management Systems
  • Web Development
  • Software Engineering
